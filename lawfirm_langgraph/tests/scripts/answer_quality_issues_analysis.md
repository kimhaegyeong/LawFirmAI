# 최종 답변 품질 문제점 분석

## 발견된 문제점

### 1. 특정 사건에만 집중하는 문제
**현상**: 
- 답변이 "대전지방법원 대전지방법원-2014가단43882"라는 특정 판례 사건의 내용을 그대로 복사한 것처럼 보임
- 특정 사건의 세부사항(피고 엘지, 피고 에스케이, 원고 등)이 상세히 나열됨

**원인**:
- 프롬프트에 "일반 법적 원칙을 먼저 설명하라"는 지시가 있지만, LLM이 이를 따르지 않음
- 특정 판례 사건의 내용을 그대로 복사하는 것을 방지하는 지시가 부족함
- 문서 내용을 그대로 복사하는 경향이 있음

**해결 방안**:
1. 프롬프트에 "특정 사건의 내용을 그대로 복사하지 마라"는 명확한 지시 추가
2. "판례는 법적 원칙 중심으로 설명하라"는 지시 강화
3. "일반 법적 원칙을 먼저 설명하라"는 지시를 더 강조

### 2. 일반적인 법적 원칙 부재
**현상**:
- "계약서 작성 시 주의사항"에 대한 일반적인 답변을 기대했지만, 특정 사건의 판결문 내용만 나열됨
- 일반적인 법적 원칙이나 주의사항이 없음

**원인**:
- 프롬프트에 "일반 법적 원칙을 먼저 설명하라"는 지시가 있지만, LLM이 이를 따르지 않음
- 답변 구조화 지시가 충분히 강력하지 않음

**해결 방안**:
1. 프롬프트에서 "일반 법적 원칙을 먼저 설명하라"는 지시를 더 강조
2. 답변 구조화 지시를 더 명확하게 제시
3. 예시를 통해 올바른 답변 형식을 제시

### 3. 구조화되지 않은 답변
**현상**:
- 일반 법적 원칙 → 주요 주의사항 → 관련 법령 → 판례/해석례 순서로 구성되어야 하는데, 그렇지 않음
- 특정 사건의 내용이 먼저 나오고, 일반적인 법적 원칙이 없음

**원인**:
- 답변 구조화 지시가 충분히 강력하지 않음
- LLM이 문서 내용을 그대로 복사하는 경향이 있음

**해결 방안**:
1. 답변 구조화 지시를 더 명확하게 제시
2. 예시를 통해 올바른 답변 형식을 제시
3. 답변 시작 부분에 일반 법적 원칙을 먼저 설명하도록 강제

### 4. 법률 인용이 불완전함
**현상**:
- "대법", "민사소송법", "지방세법", "사용방법" 등으로만 나열되어 있고, 구체적인 조문번호가 없음
- 법령 조문 인용이 불완전함

**원인**:
- 프롬프트에 "법령 조문을 구체적으로 인용하라"는 지시가 있지만, LLM이 이를 따르지 않음
- 문서에서 법령 조문을 찾지 못했을 가능성

**해결 방안**:
1. 법령 조문 인용 지시를 더 강화
2. 예시를 통해 올바른 인용 형식을 제시
3. 법령 조문 인용이 없으면 답변 품질이 낮게 평가되도록 검증 강화

### 5. 답변의 일관성 부족
**현상**:
- 첫 부분은 특정 판례 내용이고, 마지막 부분은 다른 문서(국가인권위원회)의 내용이 갑자기 나타남
- 답변의 일관성이 없음

**원인**:
- 여러 문서의 내용을 종합하지 못하고 각각을 따로 나열함
- 문서 내용을 그대로 복사하는 경향이 있음

**해결 방안**:
1. "여러 문서의 내용을 종합하여 답변하라"는 지시 추가
2. "문서 내용을 그대로 복사하지 말고, 일반적인 법적 원칙을 추출하라"는 지시 강화

## 적용된 개선 사항

### 1. 답변 구성 원칙 강화
- "일반 법적 원칙을 먼저 설명하라"는 지시를 더 명확하게 제시
- 답변 구조화 지시를 더 강화
- 특정 사건에만 집중하지 말라는 지시 추가

### 2. 문서 활용 지침 강화
- "특정 판례 사건의 내용을 그대로 복사하지 마라"는 명확한 지시 추가
- "판례는 법적 원칙 중심으로 설명하라"는 지시 강화
- "절대 금지 사항" 섹션 추가

### 3. 답변 시작 지시 강화
- "주어진 문서를 바탕으로 답변드리면:" 같은 문구를 사용하지 말라는 지시 추가
- "바로 일반적인 법적 원칙부터 설명을 시작하라"는 지시 추가
- 예시를 통해 올바른 답변 시작 형식을 제시

## 추가 개선 제안

### 1. 답변 검증 강화
- 답변에서 특정 사건의 내용이 그대로 복사되었는지 검증
- 일반 법적 원칙이 먼저 설명되었는지 검증
- 답변 구조가 올바른지 검증

### 2. 프롬프트 예시 추가
- 올바른 답변 형식의 예시 추가
- 잘못된 답변 형식의 예시 추가
- 비교를 통해 올바른 답변 형식을 명확히 제시

### 3. 답변 후처리 강화
- 답변에서 특정 사건의 내용을 제거하는 후처리 로직 추가
- 일반 법적 원칙을 추출하는 후처리 로직 추가
- 답변 구조를 재구성하는 후처리 로직 추가
